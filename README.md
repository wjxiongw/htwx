# 网站底层项目
参考网上知识林项目
Springboot+JPA+Thymeleaf 
H2数据库，运行自动创建表，初始化后自动生成权限管理，可对按钮级权限控制。
主要功能有：

此项目为网站最底层的代码，已系统权限管理部份：

1、用户管理

2、角色管理

3、菜单管理

4、自动生成系统菜单

5、为角色授权菜单

6、为用户分配角色

7、Swagger2查看系统提供API服务
